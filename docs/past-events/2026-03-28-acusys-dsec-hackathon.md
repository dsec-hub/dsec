# ACUSYS x DSEC Hackathon 2026 — Trust No One, Treat Everyone

**Dates:** Saturday–Sunday, 28–29 March 2026, 9:00 AM – 9:00 PM (36 hours)
**Venue:** VIC in-person + Online
**Trimester:** T1 2026
**Type:** Flagship
**Format:** Hybrid
**Committee:** Technical
**Partner:** ACUSYS, DUCA

## Theme

Design zero trust systems that give doctors only the patient data they need — nothing more, nothing less.

## Problem Statement

Healthcare data access is over-broad: a doctor treating a broken arm shouldn't see therapy records from 3 years ago; a nurse checking blood pressure shouldn't see the full surgical history. Build the right-data-for-the-right-person solution.

## What Participants Could Build

- **Role-Based Login** — different views for doctors, nurses, specialists
- **Smart Dashboard** — patient data filtered by treatment context
- **Alert System** — flag suspicious access patterns in real time
- **Breach Simulation** — show what goes wrong when everyone has access to everything

## Details

- Teams: 1–4 people
- Open to beginners, solo hackers, high school students
- Working prototypes score higher
